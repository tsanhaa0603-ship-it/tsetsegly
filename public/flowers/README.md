## Зөвлөмж

- Формат: `.png` (тунгалаг дэвсгэртэй бол илүү дээр) эсвэл `.jpg`
- Хэмжээ: 800×800px орчим, квадрат хэлбэртэй
- Файлын нэрийг жижиг үсгээр, зайны оронд зураас (`-`) ашиглана
- Вэбэд ашиглах зам: `/flowers/сарнай-улаан.png`
